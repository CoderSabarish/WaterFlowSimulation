# 🌊 Water Flow Simulation

A simulation-based project that models the flow of water through a pipe network using Data Structures and Graph Algorithms. The system helps determine the shortest path and analyze water distribution efficiently.

---

## 📌 Features

- 🔹 Graph-based representation of pipe networks
- 🔹 Add and manage junctions and pipe connections
- 🔹 Find the shortest path for water flow
- 🔹 Efficient route calculation using Dijkstra's Algorithm
- 🔹 User-friendly simulation interface
- 🔹 Role-based access for administrators and users

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/CoderSabarish/WaterFlowSimulation.git
```

### Navigate to the project directory

```bash
cd WaterFlowSimulation
```

### Install dependencies

```bash
npm install
```

### Start the server

```bash
node app.js
```

---

## 📖 Working Principle

1. Nodes represent junctions.
2. Edges represent pipes.
3. Dijkstra's Algorithm calculates the shortest time path.
4. Users can simulate water flow between selected nodes.
