Simulation of Water
Flow in Pipes

Visualize water flowing through pipe networks using Graph data structures and Dijkstra's Algorithm. Build networks, find shortest paths, and watch the flow animate in real-time.

🚀 Run Simulation 📚 View Applications
🔗

Graph (Adjacency List)

Pipe networks are represented as weighted undirected graphs using an adjacency list. Each edge stores length and velocity.

⚡

Dijkstra's Algorithm

Finds the shortest-time path using a priority queue. Edge weight = Time = Length ÷ Velocity for each pipe segment.

💧

Water Flow Animation

Watch animated water particles flow along the computed shortest path with glowing neon effects on an interactive canvas.
